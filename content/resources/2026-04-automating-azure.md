## Session Overview

In this session, Jane Smith covered end-to-end Azure automation using the **Az PowerShell module**.

### Topics Covered

- Authenticating securely with service principals
- Writing idempotent resource provisioning scripts
- Integrating PowerShell deployments into CI/CD pipelines
- Common pitfalls and how to avoid them
